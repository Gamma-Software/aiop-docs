# CLI (Interface en Ligne de Commande)

Les Interfaces en Ligne de Commande (CLI) sont des outils qui permettent d'interagir avec des applications en utilisant des commandes textuelles. Aiop propose une CLI pour interagir et lancer des commandes.

## Commandes

Les commandes sont des instructions qui permettent d'interagir avec l'application. Les commandes sont composées de plusieurs éléments :

- [**init**](/docs/cli/init) : La commande Init permet d’initier un projet
- [**lint**](/docs/cli/lint) : La commande Lint permet de faire des vérifications sur le projet, ces déclarations et les resources
- [**build**](/docs/cli/build) : La commande Build permet de construire le package
- [**plugins**](/docs/cli/plugins) : La commande Plugins permet de gérer les plugins
