# Concepts

Aiop livre une librarie permettant d'étendre les fonctionnalités spécifiques à chaque playbook. Cette librairie vous permettra de développer des scripts pour répondre à vos besoins spécifiques.

## Tâches et Plugins

Les tâches personnalisées sont des classes python exécutables et qui respectent une structure spécifique. Les tâches sont utilisées pour effectuer des actions spécifiques dans un playbook et interviennent dans le déroulement de la création du package. Ils peuvent pour le moment être lancés que dans l'étape de `lint` et `post-build`.

Les plugins sont des extensions de fonctionnalité de Aiop. Il sont destinés aux entreprises avec des contraintes et des demandes propre à leur métier. Les plugins sont développés par l'équipe d'ingénierie d'Aiops. C'est la société Leaptech qui détient les droits d'usages. Veuillez contacter le [support](mailto:support@aiop.io) pour toute demande de développement de tâches ou de plugins.
