Cette introduction rapide vous aide à commencer à utiliser Aiop rapidement.

## Prerequis

#### Installation de Python

Installez la version recommandée de Python 3.11.8 en utilisant votre gestionnaire de paquets. Mettez à jour votre gestionnaire de paquets et installez les dépendances nécessaires. Puis ajoutez le dépôt de deadsnakes pour installer les versions de Python.

```bash
sudo apt-get update -y
sudo apt-get upgrade -y
sudo apt-get install build-essential libssl-dev libffi-dev python3-dev
sudo add-apt-repository ppa:deadsnakes/ppa
```

Ensuite installez la version de python de votre choix. Veuillez installer les versions supérieurs à 3.6.

```bash
sudo apt install python3.6
```

En revanche nous recommandons d’installer la version 3.11.8.

```bash
sudo apt install python3.11
```

<Callout type="info" emoji="ℹ️">
  Vous pouvez toute fois installer la dernière version 3.12. En revanche nous ne garantissons pas que Aiop fonctione.
</Callout>

Vérifiez la version de python installée avec la commande `python3 --version`.

#### Installation des dépendances

Additionnellement installez des outils de développement et des bibliothèques pour Python que Aiop utilisera.

```bash
sudo apt-get install curl iputils-ping -y
```

#### Installation de Pip
Si vous n’avez pas pip d’installez, vous pouvez l’installer avec la commande suivante:

```bash
sudo apt-get install python3-pip -y
```

### Alias

Pour simplifier l’utilisation de python, il est recommandé d’aliaser les commandes `python` et `pip` à `python3` et `pip3` respectivement:

```bash
ln -s /usr/bin/python3 /usr/bin/python
ln -s /usr/bin/pip3 /usr/bin/pip
```

## Installation

<Steps>

### Créez un compte aiop

Commencez par créer un compte aiop. Si vous n’avez pas de compte, appuiyez sur le bouton ci-dessous pour en créer un.

    Créer un compte aiop
  </a>
</Button>


### Installez aiop
Installer aiop est simple. Ouvrez un terminal et exécutez la commande suivante:

```bash
pip install aiop
```

Cela installera aiop et toutes ses dépendances.


### Télécharger un modèle de playbook

Pour utiliser aiop, vous avez besoin d’un [playbook](/docs/vocabulaire#playbook). Vous pouvez installer un modèle de playbook en exécutant la commande suivante:

```bash
aiop init ./my-playbook
```

Cela téléchargera un modèle de playbook dans le dossier `my-playbook`.

### Générer sont premier package

Vous y êtes enfin ! Vous pouvez maintenant générer votre premier package en exécutant la commande suivante:

```bash
cd my-playbook
aiop build archi1/model_range/model1 --apps app1 app2
```

</Steps>

<Callout type="info" emoji="✅">
  Terminé, il est maintenant temps de modifier le playbook ainsi que vous l’approprier.
</Callout>
