# Concepts

Aiop livre une librarie permettant d'étendre les fonctionnalités spécifiques à chaque playbook. Cette librairie vous permettra de développer des scripts pour répondre à vos besoins spécifiques. Les plugins sont créé par l'équipe d'ingénierie d'Aiops et sont utilisés pour étendre les fonctionnalités de la plateforme.

Veuillez contacter le support pour toute demande de développement de scripts ou de plugins personnalisés.

## Scripts et Plugins

Les scripts sont des fichiers python executables et qui respectent une structure spécifique. Les scripts sont utilisés pour effectuer des actions spécifiques dans un playbook et interviennent dans le déroulement de la création du package. Ils peuvent pour le moment être lancé que dans l'étape de `lint` et `post-build`.

Les plugins sont des extensions de fonctionnalitée de Aiop. Il sont déstinés aux entreprises avec des contraintes et des demandes propre à leur métier. Les plugins sont développés par l'équipe d'ingénierie d'Aiops. C'est la société Leaptech qui détient les droits d'usages.
