# Foire Aux Questions

Voici les questions les plus fréquemment posées par nos utilisateurs. Si vous ne trouvez pas la réponse à votre question, n'hésitez pas à nous [contacter](mailto:support@aiop.io).

## J'aimerais savoir si Aiop est compatible avec mon système d'exploitation ?

Aiop est compatible avec les systèmes d'exploitation Windows, Mac et Linux. L’installation de Aiop diffère selon le système d’exploitation. Pour plus d’informations, veuillez consulter la documentation d’installation.

<Cards>
  <Card icon={<MonitorDown />}  title="Doc d’installation" href="https://docs.python-cerberus.org/" target="_blank" arrow/>
</Cards>

## J'amerais changer de plan d'abonnement, comment procéder ?

Pour le moment, pour changer de plan d'abonnement, vous devez contacter le support client. Nous travaillons actuellement sur une solution pour que vous puissiez changer de plan d'abonnement directement depuis votre espace client.

  <Card icon={<SendHorizontal />}  title="Contacter le support" href="mailto:support@aiop.io" target="_blank"/>
</Cards>
