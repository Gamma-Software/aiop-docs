# Conversion en pdf

Aiop est capable de convertir des ressources en pdf. Pour le moment c’est limité à des fichiers [Markdown](/docs/vocabulary#markdown), mais d’autres formats de test seront ajoutés à l’avenir.

## Depuis un Markdown

Pour déclarer une ressource Markdown à convertir en pdf, il suffit de déclarer le fichier avec l’extension `.md` et d’indiquer la destination comme étant un fichier `.pdf` dans le fichier de déclaration `.aml`.

<FileTree>
  <FileTree.Folder name="playbook" defaultOpen>
    <FileTree.Folder name=".git">
        <FileTree.File name="..." />
    </FileTree.Folder>
    <FileTree.Folder name=".aiop">
        <FileTree.File name="inventory_config.yml" />
        <FileTree.File name="inventory.yml" />
        <FileTree.File name="playbook.yml" />
    </FileTree.Folder>
    <FileTree.Folder name="resources" defaultOpen>
        <FileTree.File name=".aml" />
        <FileTree.File name="doc.md" />
    </FileTree.Folder>
  </FileTree.Folder>
</FileTree>

```yaml {2} filename="resources/.aml" /doc.pdf/
- source: doc.md
  destination: doc.pdf
  compatibility:
    - ...
```

Le package généré contiendra le fichier `doc.pdf` qui sera une conversion du fichier `doc.md` en pdf.

> La librairie utilisée pour la génération de ce pdf depuis un fichier markdown est [mdpdf](https://github.com/normanlorrain/mdpdf).
