# Applications / Librairies tièrces

## Cerberus

Cerberus est une bibliothèque de validation de données pour Python. Elle est conçue pour valider des données structurées telles que des documents JSON. Cerberus assure la validation et la normalisation de données avec une syntaxe simple et intuitive.

License: ISC License

<Cards>
  <Card title="Voir le site" href="https://docs.python-cerberus.org/" target="_blank" arrow/>
</Cards>

## Dohq Artifactory

Dohq Artifactory est une bibliothèque Python pour interagir avec Artifactory, un gestionnaire de dépôts d'artefacts. Il permet de gérer des dépôts, d'uploader, de télécharger et de supprimer des artefacts depuis Python.

License: MIT License

<Cards>
  <Card title="Voir le site" href="https://github.com/devopshq/artifactory" target="_blank" arrow/>
</Cards>

## PyYaml

PyYaml est une bibliothèque Python pour lire et écrire des fichiers YAML. Elle permet de convertir des données YAML en structures Python et de convertir des structures Python en YAML.

License: MIT License

<Cards>
  <Card title="Voir le site" href="https://pyyaml.org/" target="_blank" arrow/>
</Cards>

## Git Python

Git Python est une bibliothèque Python pour interagir avec des dépôts Git. Elle permet de gérer les dépôts Git, d'effectuer des opérations de commit, de clonage, de branches, etc.

License: BSD-3-Clause license

<Cards>
  <Card title="Voir le site" href="https://gitpython.readthedocs.io/en/stable/tutorial.html" target="_blank" arrow/>
</Cards>

## Gitignore Parser

Gitignore Parser est une bibliothèque Python pour parser les fichiers .gitignore. Elle permet d'analyser les règles de filtrage des fichiers et répertoires spécifiées dans les fichiers .gitignore.

License: MIT License

<Cards>
  <Card title="Voir le site" href="https://github.com/mherrmann/gitignore_parser" target="_blank" arrow/>
</Cards>

## Py7zr

Py7zr est une bibliothèque Python pour manipuler les fichiers 7z. Elle permet de compresser et de décompresser des archives 7z depuis Python.

License:  LGPL-2.1 license

<Cards>
  <Card title="Voir le site" href="https://py7zr.readthedocs.io/en/latest/user_guide.html" target="_blank" arrow/>
</Cards>

## MDPDF

MDPDF est un outil en ligne de commande pour convertir des fichiers Markdown en fichiers PDF. Il offre une manière simple de générer des documents PDF à partir de fichiers Markdown.

License: Apache-2.0 license

<Cards>
  <Card title="Voir le site" href="https://github.com/BlueHatbRit/mdpdf" target="_blank" arrow/>
</Cards>

## Pandas

Pandas est une bibliothèque Python pour la manipulation et l'analyse de données. Elle offre des structures de données puissantes et des outils pour le nettoyage, la transformation et l'analyse de données.

License: BSD License

<Cards>
  <Card title="Voir le site" href="https://pandas.pydata.org/" target="_blank" arrow/>
</Cards>

## Tabulate

Tabulate est une bibliothèque Python pour formater des données tabulaires en texte. Elle permet de créer des tableaux à partir de listes de données et de les afficher sous forme de texte dans la console.

License: MIT License

<Cards>
  <Card title="Voir le site" href="https://github.com/astanin/python-tabulate" target="_blank" arrow/>
</Cards>

## Click

Click est une bibliothèque Python pour la création d'interfaces en ligne de commande (CLI). Elle permet de définir des commandes, des options et des arguments pour créer des applications en ligne de commande de manière simple et intuitive.

License: BSD License

<Cards>
  <Card title="Voir le site" href="https://click.palletsprojects.com/en/8.1.x/" target="_blank" arrow/>
</Cards>

## Rich

Rich est une bibliothèque Python pour la mise en forme de texte en console. Elle offre des fonctionnalités avancées pour styliser et mettre en forme le texte, les tableaux et les barres de progression dans la console.

License: MIT License

<Cards>
  <Card title="Voir le site" href="https://rich.readthedocs.io/en/stable/introduction.html" target="_blank" arrow/>
</Cards>

## Pydantic

Pydantic est une bibliothèque Python pour la validation de données et la configuration. Elle permet de définir des modèles de données avec des types et des contraintes, et de valider les données par rapport à ces modèles.

License: MIT License

<Cards>
  <Card title="Voir le site" href="https://docs.pydantic.dev/latest/" target="_blank" arrow/>
</Cards>

## Simple Term Menu

Simple Term Menu est une bibliothèque Python pour créer des menus textuels interactifs en console. Elle permet de définir des options et des actions associées à chaque option pour créer des menus interactifs dans la console.

License: MIT License
